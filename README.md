# RC CAR

This is project created for my bachelor thesis. It is about active drive of a 1/10 RC Car model. 
This project includes all the source code that is located in the control unit Arduino Due. It contains set up sequence and then the main loop where teh active control takes place.

### Prerequisites

The project is created in Atmel Studio 7 (it can be downloaded from http://www.microchip.com/mplab/avr-support/atmel-studio-7). 
You need to add there an extension named Visual micro in order to build it properly.
